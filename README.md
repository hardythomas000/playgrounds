## Features

- **Zero dependencies** — no frameworks, no CDNs, no build tools
- **Single file** — each playground is one self-contained HTML file
- **Mobile-first** — touch-friendly controls, responsive layouts
- **60fps** — canvas-based rendering with requestAnimationFrame
- **Light theme** — clean, readable on any device

## Usage

Open any `.html` file directly in a browser, or visit the [GitHub Pages site](https://hardythomas000.github.io/playgrounds/).
